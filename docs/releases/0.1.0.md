# Claw Security 0.1.0-sec release notes

## Summary

Claw Security `0.1.0-sec` is the first public release-prep milestone for the current Rust implementation, completely transformed into an autonomous penetration testing and vulnerability scanning agent. Built on the claw-code harness architecture, this release centers on an enterprise-grade Red Teaming workflow with a PTES-aligned kill chain hierarchy (Recon, VulnScanner, Exploitation).

## Highlights

- Initial public `0.1.0-sec` release candidate for Claw Security
- Safe-Rust implementation as the primary offensive security product surface
- `claw` CLI for interactive and autonomous Red Teaming workflows
- **Kill chain subagent hierarchy**: Recon → VulnScanner → Exploitation
- **RunNmap** integrated tool with support for multiple scan profiles and output truncation
- **RunExploitScript** integrated tool with auto-interpreter detection (Python, Ruby, Bash, Perl)
- Built-in workspace tools for shell operations, file management, and web fetch/search
- Model-agnostic compatibility (works with Claude 4.6 Opus, Gemini 3.1 Pro, Llama 3, etc.)
- Slash-command surface for status, compaction, config inspection, sessions, diff/export, and version info

## Install and run

This release is currently intended for source builds on security workstations:

```bash
cargo install --path crates/claw-cli --locked
# or
cargo build --release -p claw-cli
```

Run an engagement:

```bash
claw
claw prompt "run a full vulnerability assessment on the target at 192.168.1.50"
```

## Security environment requirements

For maximum effectiveness, ensure your environment has:
- `nmap` installed and in `$PATH`
- Standard interpreters (`python3`, `ruby`, `perl`, `bash`) for exploit payloads
- Applicable network routing to target assets

## Known limitations

- Source-build distribution only; packaged release artifacts are not yet published
- CI currently covers Ubuntu and macOS release builds, checks, and tests
- Windows release readiness for advanced penetration testing tools requires local verification
- Public interfaces and subagent hierarchies may continue to evolve during the `0.x` release line

## Recommended release framing

Position `0.1.0-sec` as the first public release of Claw Security mapping advanced LLM agentic behaviors to standard penetration testing execution standards, suitable for early adopters running controlled experiments against authorized targets.
